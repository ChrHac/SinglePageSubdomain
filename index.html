Selection Sort
Show Algorithm Hide Algorithm

Selection sorting is an algorithm that takes an array of numbers, finds the smallest value and swaps it with the element in first position. It then repeats this process removing the first element of the array as it has already been sorted.

Before sorting: arr = [18,15,3,8,7,16,14,17,6,19]

Insertion Sort
Show Algorithm Hide Algorithm

Insertion sorting is an algorithm that takes an array of numbers, iterates through the elements comparing each to it's predecessor. If the element is smaller than it's predecessor then it is moved to the first position of the array with the other elements moving 1 spot forward to make room.

Before sorting: arr = [12,13,11,7,4,5,19,17,15,20]

Quick Sort
Show Algorithm Hide Algorithm

Quick sort takes a pivot point, most commonly the middle point of the array and compares elements on either side of it to the pivot. It takes the first element on the left that is smaller than the pivot and the first element on the right that is greater than the pivot and swaps them. It then breaks the array at the pivot point and repeats the process on both sides of the pivot. This is iterated until the array has been sorted.

Before sorting: arr = [3,9,6,17,2,5,18,20,12,19]

Heap Sort
Show Algorithm Hide Algorithm

Heap sort finds the maximum of an array and moves it to the end of the array, repeating the process iteratively by removing the last element of the array from the sort.

Before sorting: arr = [19,20,11,3,5,12,18,10,6,15]
